# Beauty Bestie 💕

Welcome to **Beauty Bestie** — a friendly and fun beauty advice website designed to share tips, tricks, and inspiration for skincare, makeup, and hair care.

## 💄 About the Website
Beauty Bestie is a simple HTML and CSS project created to practice web development skills. It features a colourful background, a cute aesthetic design, and a clean layout.

## 🌸 Features
- Cute and aesthetic design
- Custom background image
- Stylish heading with Google Fonts
- Beginner-friendly HTML & CSS structure

## 💕 Purpose
This project was created to learn how to:
- Build a basic website using HTML
- Style a webpage using CSS
- Upload a project to GitHub

## 📁 Files Included
- `index.html` – Main website file
- `salon Chat gpt.png` – Background image

## 🚀 How to View
1. Download or clone this repository
2. Open `index.html` in any web browser
